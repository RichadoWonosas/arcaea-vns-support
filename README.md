Visual Studio Code's extension for arcaea visual novel script (.vns) file's reading and editing

- [x] Syntax Highlight
- [x] Basic Snippet Autocompletion
- .vns File Inspection
  - [x] .vns File Parsing
  - [x] Basic Value Check
  - [ ] File Usage Check for Image and Music
- [ ] IntelliSense Supports
